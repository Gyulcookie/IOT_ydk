# DB 연동
```
.
+-- WebContent
|   +-- main    - sample.jsp
|   +-- include - include.jsp (db 연결 관련)
|               - topmenu.jsp (메뉴와 로그)
|   +-- css     - layout.css
|   +-- images
|   +-- board
|   +-- member
|       +-- scoreWrite.jsp
|       +-- scoreWriteSave.jsp
|       +-- scoreList.jsp
|       +-- scoreModify.jsp (수정화면)
|       +-- scoreModifySave.jsp (수정처리)
|       +-- scoreDelete.jsp (삭제)
|   +-- js
|   +-- index.jsp -> 파일
```

- scoreWriteSave.jsp 에 있던 db 관련 코드 include.jsp로 이동
- menu 관련 코드 topmenu.jsp
